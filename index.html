DOM Manipulation with Forms

The form includes fields for capturing personal information such as first name, last name, address, pincode, gender, choice of food, state, and country. The form utilizes Bootstrap classes for layout and styling to ensure responsiveness across various device sizes. Upon submission, the form data is dynamically added to a table below the form. The table displays the submitted data with columns corresponding to the form fields. Additionally, JavaScript is included to handle form submission, data processing, and table population
